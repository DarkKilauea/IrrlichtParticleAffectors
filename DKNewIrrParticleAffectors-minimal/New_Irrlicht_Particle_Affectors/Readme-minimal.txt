1. How to start
==========================================================================

Unzip this into the top level of your 1.4.1 irrlicht directory.
Simply recompile the library to enable use of the enhancements.

Alternatively, you can apply the supplied diff and recompile.

You can find all the affectors inside the New_Irrlicht_Particle_Affectors
directory.

==========================================================================
2. Requirements
==========================================================================
You will need irrlicht 1.4.1.  This has not been tested with any other irrlicht
version!

You will also need a compatible compilier.  See Irrlicht's readme for details.
