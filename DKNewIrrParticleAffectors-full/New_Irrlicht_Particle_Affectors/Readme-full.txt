1. How to start
==========================================================================

1. Unzip this into the top level of your 1.4.1 irrlicht directory.
2. Add the new files to your IDE project or Makefile.
3. Recompile the library to enable use of the enhancements and affectors.

Alternatively, you can apply the supplied diff and recompile.

==========================================================================
2. Requirements
==========================================================================
You will need irrlicht 1.4.1.  This has not been tested with any other irrlicht
version!

You will also need a compatible compilier.  See Irrlicht's readme for details.
